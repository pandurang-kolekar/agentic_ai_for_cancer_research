# Datasets

This folder contains sample datasets and instructions for accessing public datasets used in this book.

## Contents

### Sample Datasets
Sample data files for hands-on exercises (when available):
- `chapter_01/` - Sample data for Chapter 1
- `chapter_02/` - Sample data for Chapter 2
- etc.

### How to Access Public Datasets

Most chapters use publicly available data. See individual chapter READMEs for specific datasets.

## Dataset Guidelines

### Size Limitations
- Sample datasets included: < 10 MB
- Large datasets: Download links provided
- Very large datasets: Cloud access instructions provided

### Data Formats
- Genomic: FASTA, VCF, BAM (aligned sequences)
- Tabular: CSV, TSV (gene expression, variants)
- Biological Networks: GML, GraphML (interaction networks)

### Privacy and Ethics
All data is publicly available or de-identified research data.
